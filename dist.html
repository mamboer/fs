Distribution

This utility is primarily distributed as an npm package (simply npm install and use it in your Node.js development environment).

In addition, the GitHub Releases Page contains a tarball for each release (fs_{ver}tar.gz) ... which promotes various executable bindings and documentation (should you wish to retain the docs locally). The tarball contains the following executable bindings (it's your choice):

File Or          Module
Directory  What  Bindings Notes
=========  ====  ======== ======================================
libs/      ES6   ES       the master source
index.js   ES5   ES       the main entry file
docs/      HTML  N/A      documentation
