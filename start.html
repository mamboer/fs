Getting Started

Getting Started

Install

npm install --save @aotu/fs

Or with yarn

yarn add @aotu/fs

Access

const { walk, flatten } = require('@aotu/fs');
// -OR-
const FS = require('@aotu/fs');

walk(...)
flatten(...)
// -OR-
FS.walk(...)
FS.flatten(...)
